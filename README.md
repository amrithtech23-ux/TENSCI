# ⚖️ 10th Standard TN State Board Science Chatbot

An academic-focused Streamlit chatbot built on the **Qwen** model (via OpenRouter API), specifically tailored for **10th Standard Tamil Nadu State Board Science** students. Covers Laws of Motion, Optics, and core curriculum topics.

## 📦 Features
- 📘 Academic tone aligned with TN State Board syllabus
- 🎯 10 randomly rotating curriculum-specific prompts
- 🎨 Custom UI: Bold borders, blue backgrounds, white bold text
- 🔄 Reset functionality & session state management
- 🔒 Secure API key handling via `secrets.toml`
